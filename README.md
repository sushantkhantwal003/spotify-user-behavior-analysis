🎧 Spotify User Behavior Analysis

📌 Project Overview
This project analyzes a dataset of 50,000 Spotify users to uncover patterns in user behavior, engagement, subscription trends, and platform usage. The analysis is performed using Python (Pandas, NumPy) and exploratory data analysis (EDA) techniques.
The goal is to derive data-driven insights without assumptions and evaluate how different factors influence user behavior.

🛠️ Tools & Technologies Used
Python 🐍
Pandas & NumPy
Matplotlib & Seaborn
Jupyter Notebook

📂 Dataset Details
Total Records: 50,000
Features include:
  Subscription Type (Free / Premium)
  Country
  Genre Preferences
  Device Usage
  Listening Hours
  Skip Behavior
  Ad Interaction
  Age Group
  Recommendation Ratings
  Churn Status

🔍 Key Analysis Performed
1. Subscription Distribution
  ~50% Free vs ~50% Premium users
  Indicates a balanced monetization model
2. Geographic Distribution
  Users evenly spread across countries
  No dominant region
3. Genre Popularity
  All genres show uniform preference
  No dependency on specific content
4. Churn Analysis
  ~33% users churned
  Indicates moderate user retention challenge
5. Device Usage
  Equal distribution (~20% each) across:
  Mobile, Desktop, Tablet, Smart Speaker, Car System
6. Listening Behavior vs Skips
  Correlation ≈ 0
  Listening time and skipping behavior are independent
7. Ad Interaction vs Subscription
  No strong relationship found
  Ads do not significantly drive premium conversion
8. Listening Hours Distribution
  Uniform spread across users
  No heavy/light user segmentation
9. Age vs Engagement
  Equal engagement across age groups
10. Recommendation Ratings
  Even distribution (1–5)
  Indicates neutral user satisfaction

🧠 Key Insights
All variables show near-perfect uniform distribution
No strong correlations or trends observed
Suggests dataset is likely:
  Synthetic or evenly distributed
Real-world datasets typically show:
  Skewness
  Clustering
  Behavioral patterns

📊 Business Implications
No single factor drives user behavior
All segments contribute equally
Limited scope for targeted strategies based on this dataset
Highlights importance of real-world data variability
